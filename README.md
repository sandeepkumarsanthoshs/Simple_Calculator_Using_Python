This Simple Calculator Project is built using the Python Tkinter package and consists of basic arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## How to Run

1. **Download the File**
   - Clone the repository or download the ZIP file from the repository.

2. **Open it on any Python Supported IDE**
   - Recommended: Pycharm

3. **Run the Project**
   - Open the project in the IDE.
   - Run the project to use your calculator.

## Prerequisites
- Python installed on your system
- Python Integrated Development Environment (IDE) such as Pycharm

## Usage
Once the project is running, the calculator window will appear. You can use the buttons to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.
